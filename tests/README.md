# AI Test Suite

Author:
Samantha × ChatGPT

Purpose

所有 AI 功能修改後，

必須利用這些案例測試。

不能因為增加新功能，

反而讓原本正確的判斷變錯。

---

## Test Categories

- Fake Breakout
- Event Intelligence
- Recovery AI
- Discovery AI
- Winner DNA
- AI Consensus
- Portfolio Analysis

---

## Rule

每完成一個版本，

至少測試所有案例一次。

若結果與預期不同，

不得 Release。
